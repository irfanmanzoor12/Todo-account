# Accounting Ledger App — Project Overview

## Purpose
An accounting ledger app that evolves from console app to AI chatbot.

## Current Phase
Phase II — Full-Stack Web Application

## Tech Stack
- Frontend: Next.js 14 (App Router), TypeScript, Tailwind CSS, Better Auth
- Backend: Python FastAPI, SQLModel, Neon Serverless PostgreSQL
- Auth: Better Auth with JWT tokens
- Deployment: Frontend on Vercel, Backend on local/cloud

## Features
- [x] Phase I: Console App (Debit/Credit/Expense/Income)
- [ ] Phase II: Web App with Auth + Persistent DB
- [ ] Phase III: AI Chatbot
